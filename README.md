学习任何一种语言，不能都按固定的语法死记硬背。应当像学习母语一样慢慢练习。  
首先要做的是多看，看一些视频教程，哪怕最简单的视频教程。  
通过视频教程，我们知道了东西长什么样，发音。  
不断重复看，听这个过程。如果感觉难那可以跟着说。  
看见样子，听到声音，跟着说，同时阅读相关单词或句子。
看+听+说+读。  
以语句块为单位，重复。参照视频解说。
找出生活中常用物品，常用语句，常用高频词去练。  
练的时候也要注意，要全部使用所学的语言。如学英语，那所有的资料都用英语的。  
这里我们学习的是法语，所有资料都是用法语描述的。  

重点1语言分段记高频，用看+听+说+读提升。  
重点2学哪种语言只用那种语言描述，不要翻译成你的母语。
